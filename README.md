Проект представляет собой Android-приложение, демонстрирующее полный набор возможностей базовых и интерактивных оповещений (Notifications). 
Реализованы простые уведомления, уведомления для открытия активности, уведомления с кнопками и текстовым ответом, оповещения с уникальной вибрацией, а также отправка команд сервису через BroadcastReceiver. 
Приложение работает на Kotlin и использует современную систему Notification Channels, PendingIntent-ы, RemoteInput, кастомную вибрацию и работу с сервисами.
Главная активность содержит набор кнопок — каждая запускает отдельный сценарий уведомления: простое уведомление, уведомление с переходом в приложение, оповещение, управляющее сервисом, lockscreen-уведомление и интерактивное уведомление с кнопкой и inline-reply. 
Реализован запрос runtime-разрешения POST_NOTIFICATIONS на Android 13+. 
Очистка оповещений выполняется через NotificationManager.cancelAll().
